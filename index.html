CafÉ Myriad Trendy Website · 2014

The goal was to create a website that encompassed popular web design trends of 2014. I started from scratch with a few basic ideas and built the site in the browser through multiple iterations.

The site features a large parallax hero image with simple Open Sans font-styling. Below that, flat icons introduce a three column layout with the café's mantra. Overall, the site is responsive and featuers a simple drop-down navigation when the user leaves the introductory hero image.

Forms Fully Responsive & Interactive

Nobody likes to fillout web forms when they're browsing through websites and e-commerce sites. I started this project to grasp every detail of web forms through practice and creativity. It is essentially a way for me to create forms that are aesthetically pleasing while trying to lower bounce rates and come up with new CSS tricks.

Each form is interactive and unique from the rest. They all have a similar HTML skeletion, but vastly different CSS. The styling ranges from Material Design to simple Flat Design.

CodeCore Twitter Bootstrap Framework

CodeCore is a programming school based in Vancouver, B.C. that provides intensive bootcamps for people that want to learn how to program. It's a full time course load of 40 hours per week with extra time alloted for assignmetns and group projects. It's run by Tammam Kbeili and his team of high quality teachers.

I was introcuded to the program by my brother and through their website. I enjoyed the simplicity of their website and wanted to recreate it. It turns out it was built using the Twitter Bootstrap Framework . I had heard lots about it and similar frameworks such as Zurb's Foundation , so I decided to learn it and implement it in my replica of CodeCore.ca. My CodeCore project had turned into a full Bootstrap replication of the original website.

View Project

3D CSS Drawing XYZ, Animation, Transform

I've always enjoyed making CSS drawings to help myself learn everything there is to know about the Box Model, positioning, border-radius, and more. I decided to implement the same technique but make it 3D as a challenge. This would force me to understand CSS transforms, perspective, and the animation property.

During the beginning I had trouble grasping the three dimensional planes and how to move objects around them. By the end, I had a strong grasp over the rotation and movement of objects in the X, Y, and Z planes. Also, I now fully understand the perspective property along with the animation property. If you haven't already, hover of the Robot while it is rendered in 3D to reveal what is inside.

Caveat: only works smoothly in the most recent versions of Chrome and Safari.

McGill University B.A. Psychology / Music Technology

I attended McGill University from 2010-2014 where I majored in Psychology and minored in Musical Applications of Technology. Psychology has always intrigued me since I have always wondered why people do things, what motivates them, and how the brain functions.

My first exposure to web development was through MUMT 301, Music and the Internet. We had a competition to see who could build the best website for bonus credits and the professor said to take a few hours on it - I spent 5 days. The following semester, I took COMP 202, Foundations of Programming , which covered Java and Object Oriented Programming.
